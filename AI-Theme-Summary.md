# AI-Inspired Portfolio Theme

## Color Palette

### Primary Colors
- **AI Purple**: `#6C63FF` - Main brand color for buttons, links, and accents
- **Cyan**: `#00C2FF` - Secondary color for highlights and gradients
- **Dark Background**: `#0B0C10` - Deep dark background
- **Light Gray Text**: `#E6E6E6` - Primary text color

### Gradient Colors
- **Primary Gradient**: `linear-gradient(135deg, #6C63FF 0%, #4F46E5 100%)`
- **Secondary Gradient**: `linear-gradient(135deg, #00C2FF 0%, #0EA5E9 100%)`
- **AI Accent Gradient**: `linear-gradient(90deg, #6C63FF 0%, #00C2FF 100%)`
- **Multi-color AI Gradient**: `linear-gradient(135deg, #6C63FF 0%, #8B5CF6 50%, #00C2FF 100%)`

## Key Features Applied

### Typography
- **Primary Font**: Inter, Poppins (clean, modern)
- **Display Font**: Space Grotesk, Inter (futuristic headings)
- **Monospace**: JetBrains Mono, Fira Code (code elements)

### Visual Effects
- **Neon Glow**: Hover effects on buttons and cards with AI purple/cyan glow
- **Glass Morphism**: Backdrop blur effects with subtle transparency
- **Gradient Text**: Animated gradient text for headings and key elements
- **Aurora Background**: Subtle animated background gradients
- **Card Hover Effects**: Scale, glow, and shadow animations

### Animations
- **Gradient Flow**: Animated gradient text effects
- **AI Glow**: Pulsing glow effects for interactive elements
- **Aurora**: Background animation for hero sections
- **Neon Pulse**: Text shadow animations for emphasis
- **Floating**: Smooth floating animations for decorative elements

### Component Styles
- **Buttons**: Gradient backgrounds with glow effects and shimmer animations
- **Cards**: Glass morphism with AI-themed borders and hover effects
- **Navigation**: Subtle glow effects with gradient underlines
- **Hero Section**: Dark gradient background with aurora animations
- **Forms**: Glowing focus states with AI-themed styling

## Usage Examples

### Button Classes
- `.btn-primary` - AI purple gradient button with glow effects
- `.btn-secondary` - Cyan gradient button with hover animations
- `.btn-outline` - Transparent button with AI-themed borders

### Card Classes
- `.modern-card` - Standard AI-themed card with glass morphism
- `.ai-card` - Enhanced card with glow effects
- `.project-card` - Portfolio cards with advanced hover animations

### Text Classes
- `.gradient-text-ai` - Animated AI gradient text
- `.neon-text` - Glowing text with neon pulse animation
- `.gradient-text` - Standard gradient text effect

### Background Classes
- `.hero-bg` - AI-themed hero background with aurora animation
- `.ai-glow` - Elements with AI glow animation
- `.glass` - Glass morphism effect

The theme maintains all existing content and structure while providing a modern, professional AI agency aesthetic with smooth animations and futuristic visual effects.
